# QA CHECKLIST

## Layout

- Mobile homepage has no horizontal overflow.
- Desktop homepage renders correctly.
- Navigation works.
- Footer links work.
- Images load.

## Forms

- Empty email shows an error.
- Invalid email shows an error.
- Valid email redirects to thank-you page.
- Thank-you page links to download.
- El Remanente form uses correct source tag.

## SEO

- Page titles exist.
- Meta descriptions exist.
- Canonicals exist.
- Sitemap exists.
- Robots file exists.

## Accessibility

- Keyboard navigation works.
- Focus states are visible.
- Form labels are present.
- Alt text exists for images.
- Text contrast is readable.

## Canon

- Watchman remains hooded.
- Archetypes are not superheroes.
- Enemies are not shallow monsters.
- Visuals avoid cyberpunk, superhero, cartoon, and neon aesthetics.
- Copy avoids self-help cliches and corporate softness.

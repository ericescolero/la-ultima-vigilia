# DEPLOYMENT

## Local Preview

```bash
npm run dev
```

Open:

```text
http://127.0.0.1:4173
```

## Static Hosting

Recommended hosts:

- Cloudflare Pages
- Netlify
- Vercel

Build command for current static version:

```bash
npm run build
```

Publish directory:

```text
website
```

## Production Checklist

- Domain connected.
- SSL active.
- Sitemap accessible.
- Robots file accessible.
- PDF delivery tested.
- Email provider connected.
- Analytics verified.
- Mobile QA complete.
- Canon review complete.

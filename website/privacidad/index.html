Privacidad

Politica de privacidad

Ultima actualizacion: 17 de junio de 2026. Esta pagina es una base operativa para el MVP publico y debe revisarse legalmente antes de escalar a pagos, membresias o cursos.

La Ultima Vigilia recopila correos electronicos cuando una persona solicita recursos, actualizaciones o informacion sobre El Remanente.

Uso de datos

Los datos se usan para entregar recursos solicitados, enviar contenido relacionado con la mision y comunicar futuras oportunidades de comunidad, cursos o productos.

Proveedor de email

El sistema de suscripcion esta preparado para MailerLite. Cuando un visitante envia su correo, MailerLite puede procesar la suscripcion, aplicar etiquetas de interes y enviar la automatizacion correspondiente.

Control

Cada correo enviado por el proveedor de email debera incluir una opcion para cancelar suscripcion. No se deben vender datos personales.

Herramientas futuras

El sitio podra usar analitica, proteccion contra spam y servicios de email. Cuando esas herramientas se activen, esta politica debe actualizarse con los proveedores especificos.

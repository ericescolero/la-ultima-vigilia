PROMPT STRUCTURE SYSTEM
Every major image prompt should contain:
1. SUBJECT
Who or what is being shown.
2. EMOTIONAL ENERGY
What the image should emotionally feel like.
3. ENVIRONMENT
Where the scene takes place.
4. LIGHTING
How the world is illuminated.
5. COMPOSITION
How the scene is framed.
6. ATMOSPHERIC ELEMENTS
Fog, ash, rain, smoke, snow, etc.
7. TEXTURE LANGUAGE
Realistic textures and cinematic realism.
8. SYMBOLIC DETAILS
Subtle metaphorical elements.
9. NEGATIVE PROMPT
What to avoid.

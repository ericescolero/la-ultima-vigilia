# COURSE ARCHITECTURE

## La Ultima Vigilia

Version: 1.0

---

## Purpose

Courses exist to turn conviction into formation.

They should help men:

- See clearly.
- Develop discipline.
- Resist deception.
- Strengthen faith.
- Master themselves.
- Build with purpose.
- Carry responsibility.

Courses should not feel like generic self-improvement products. They should feel like structured training for the invisible war.

## Recommended Course Flow

Each course should follow this movement:

```text
Diagnosis
→ Conviction
→ Doctrine
→ Discipline
→ Practice
→ Accountability
→ Review
→ Next mission
```

This keeps the course from becoming passive content consumption.

## Course Page Requirements

Public course landing page:

- Title and subtitle.
- Who it is for.
- The battle it addresses.
- Outcomes.
- Modules overview.
- Instructor/mission note.
- CTA to enroll, join waitlist, or enter email funnel.

Private lesson page:

- Lesson title.
- Content/video/audio.
- Reflection prompt.
- Assignment.
- Resource downloads.
- Completion button.
- Next lesson.

## Canon Review Checklist

Before publishing a course:

- It is Christian and spiritually grounded.
- It forms discipline, not just knowledge.
- It avoids generic self-help.
- It avoids entertainment-first teaching.
- It uses Watchman Universe symbolism as recognition, not escapism.
- It leads to concrete action.
- It preserves the mission of awakening.
